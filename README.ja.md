# 🏋️‍♂️BMI計算機 アプリ

**Java** を使用して **Android Studio** で開発された、シンプルで **使いやすい BMI（Body Mass Index：体格指数）計算アプリ** です。
このアプリは、ユーザーの身長と体重から BMI を計算し、健康状態に関する情報を提供します。

![BMI計算機](https://github.com/KrishBharadwaj5678/BMI-Calculator/raw/main/BMICalculator.png)

## ✨ 特徴

| 機能              | 説明                      |
| --------------- | ----------------------- |
| 🧍 身長と体重の入力     | 身長と体重を簡単に入力可能           |
| 📊 BMI 計算       | BMI（体格指数）を瞬時に計算         |
| 🩺 健康状態の表示      | BMI カテゴリーを表示            |
| 🎨 ユーザーインターフェース | シンプルで直感的な UI            |
| ⚡ 軽量アプリ         | 高速でレスポンスの良い Android アプリ |
