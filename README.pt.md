# 🏋️‍♂️ Aplicativo Calculadora de IMC

Um aplicativo Android simples e **fácil de usar para calcular o IMC (Índice de Massa Corporal)**, desenvolvido com **Java** no **Android Studio**.
Este aplicativo ajuda os usuários a calcular seu IMC com base no peso e altura, além de fornecer informações sobre o estado de saúde.

![BMICalculatorDemo](https://github.com/KrishBharadwaj5678/BMI-Calculator/raw/main/BMICalculator.png)

## ✨ Funcionalidades

| Funcionalidade           | Descrição                                           |
| ------------------------ | --------------------------------------------------- |
| 🧍 Inserir peso e altura | Digite facilmente seu peso e altura                 |
| 📊 Calcular IMC          | Calcula instantaneamente o Índice de Massa Corporal |
| 🩺 Estado de saúde       | Exibe a categoria do IMC                            |
| 🎨 Interface do usuário  | Interface limpa, simples e intuitiva                |
| ⚡ Leve e rápido          | Aplicativo Android rápido e responsivo              |
