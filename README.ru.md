# 🏋️‍♂️ Приложение BMI Calculator

Простое и **удобное приложение для расчёта ИМТ (индекса массы тела)** для Android, созданное с использованием **Java** в **Android Studio**. Это приложение помогает пользователям рассчитывать свой ИМТ на основе веса и роста, а также предоставляет информацию о состоянии здоровья.

![BMICalculatorDemo](https://github.com/KrishBharadwaj5678/BMI-Calculator/raw/main/BMICalculator.png)

## ✨ Возможности

| Функция                       | Описание                                        |
| ----------------------------- | ----------------------------------------------- |
| 🧍 Ввод веса и роста          | Удобный ввод данных о весе и росте              |
| 📊 Расчёт ИМТ                 | Мгновенный расчёт индекса массы тела            |
| 🩺 Состояние здоровья         | Определение категории ИМТ                       |
| 🎨 Пользовательский интерфейс | Чистый, простой и интуитивно понятный интерфейс |
| ⚡ Лёгкое приложение           | Быстрое и отзывчивое Android-приложение         |
